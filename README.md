# Nexum

Landing page desenvolvida com HTML e CSS puro, baseada no projeto ExperTI do SENAI Frederico Jacob.

## Sobre o Projeto

O **Nexum** é uma landing page de vendas criada para apresentar o projeto **ExperTI** — uma iniciativa do SENAI voltada para atrair jovens para a área de tecnologia. A página apresenta o mercado de TI, os pilares do projeto e os cursos disponíveis.

## Seções

- **Cursos** — Dados do mercado de TI e botão de acesso aos cursos gratuitos do SENAI
- **ExperTI** — Apresentação do projeto e diferenciais do SENAI
- **Áreas do Futuro** — Os 4 pilares: Redes, Nuvem, IA e Cibersegurança

## Tecnologias

- HTML5
- CSS3 (Flexbox, Container Queries, variáveis de cor)

## Como usar

Clone o repositório e abra o `index.html` no browser:
```bash
git clone https://github.com/Erickhitman/nexum.git
cd nexum
```
